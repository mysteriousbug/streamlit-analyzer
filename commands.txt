2. This is the refined problem_tickets.xlsx. i have also put an example row below the column.
number	sys_created_on	closed_at	cmdb_ci	priority	impact
XXX1111	5/1/2024 20:12	8/1/2024 8:57		5	5 - No Impact / Single User


number - problem ticket number, which is an alphanumeric string
sys_created_on - timestamp when the ticket was created. date format could be either like '15-07-2024 06:28:20' [dd-mm-yyyy] or '12/6/2024  10:37:11 PM' [mm/dd/yyyy]
closed_at - timestamp when the ticket was closed. date format could be either like '15-07-2024 06:28:20' [dd-mm-yyyy] or '12/6/2024  10:37:11 PM' [mm/dd/yyyy]
cmdb_ci - the cmdb_ci column is the exact 'Configuration Item' that we talked about.
priority - could be any integer between 1 and 5
impact - could be one of the four values 
5 - No Impact / Single User
4 - Minor / Localized
3 - Moderate / Limited
2 - Significant / Large
